# Order & Payment Microservices (gRPC + Contract-First)

## Overview

This project is a microservices system consisting of:

- **Order Service** (REST API, Gin)
- **Payment Service** (gRPC server)
- **Proto Contracts Repository (contract-first approach)**
- **Auto-generated Go gRPC code via GitHub Actions (CI)**

The system demonstrates:
- Clean Architecture
- gRPC communication
- Idempotent order creation
- Payment authorization flow
- Contract-first development workflow
